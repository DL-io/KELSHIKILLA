# POLY-SHORE: Premium Prediction Market Trading Platform

POLY-SHORE is an enterprise-grade, proprietary autonomous trading platform for prediction markets (Polymarket & Kalshi). It provides professional-level intelligence, risk management, and execution capabilities for serious traders.

## Key Features

- **Intelligence Engine**: Multi-LLM ensemble (Ollama, Groq, OpenAI, Claude) for advanced probability estimation and sentiment analysis.
- **Robust Risk Architecture**: Hierarchical risk management with granular killswitches, adaptive position sizing via fractional Kelly, and pre-trade simulation mode for risk analysis.
- **Full Transparency**: Real-time operational dashboard with equity tracking, risk exposure gauges, and trade auditing.
- **Deployment-Ready**: Optimized for controlled, secure deployments (Railway, cloud, or dedicated hardware).
